10 - A game of numbers - ABANDONED PROJECT

Implemented in a variety of JavaScript frameworks and compile-to-languages

I wanted to implement this game in a variety of languages and frameworks, but I lost interest after doing the ES6 version. If I ever get back to this project, I'll give ClojureScript or Elm or any of the other compiles-to-js languages a try. But for now this project is on a hiatus.

The implementations are not bug-free, but at this point I don't want to dive deeper to fix those as I want to concentrate on other, more fun projects. If you want to, feel free to fix the bugs and send a pull request.
